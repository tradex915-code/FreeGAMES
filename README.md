# 🎮 FreeGAMES

Üdvözlünk a FreeGAMES-ben! Ez egy olyan platform, ahol ingyenesen elérhető játékokkal lehet kedveskedni a közösségnek.

## ⚠️ Fontos Disclaimer

Ez a projekt **oktatási és archívum célokra** készült. A feltöltött játékokat a saját felelősségünkre osztjuk meg. Kérjük, tartsd tiszteletben az egyes játékok jogszerzői jogait és a különféle szabályozásokat.

**Megjegyzés:** Ha te vagy a szerzője valamely játéknak és szeretnéd, hogy eltávolítsuk azt, vedd fel velünk a kapcsolatot.

## 🕹️ Elérhető Játékok

Jelenleg a következő játékokkal dolgozunk:

- **Among Us** - Windows, Linux
- **Terraria** - Windows, Linux
- **Forza Horizon** - Windows, Linux
- **Teardown** - Windows, Linux

A teljes listát és letöltési linkeket a [GAMES.md](GAMES.md) fájlban találod.

## 📥 Hogyan töltsem le a játékokat?

1. Menj az [GAMES.md](GAMES.md) fájlra
2. Válaszd ki a játékot, amit szeretnél
3. Kattints a letöltési linkre
4. Kövesd a telepítési utasításokat

## 🛠️ Telepítés után

Minden játékhoz van saját README a `games/[játéknév]/` mappában, amiben megtalálod:
- Rendszerre vonatkozó követelmények
- Telepítési utasítások
- Bővebb információk
