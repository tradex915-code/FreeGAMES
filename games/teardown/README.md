## 📥 Letöltés

Letöltési linkeket a [downloads.txt](downloads.txt) fájlban találod.

## 🖥️ Rendszerre vonatkozó követelmények

### Windows
- **OS:** Windows 7 SP1 64-bit vagy újabb
- **Processzor:** Intel i5-7600K vagy AMD Ryzen 5 1600
- **RAM:** 8 GB
- **Videokártya:** NVIDIA GTX 1070 vagy AMD RX 480 (4GB VRAM)
- **Tárhely:** 8-10 GB SSD
- **DirectX:** Version 11

### Linux
- **OS:** Ubuntu 18.04 LTS vagy újabb
- **Processzor:** Intel i5-7600K vagy AMD Ryzen 5 1600
- **RAM:** 8 GB
- **Videokártya:** NVIDIA GTX 1070 vagy AMD RX 480 (4GB VRAM)
- **Tárhely:** 8-10 GB SSD

## 🎮 Mit kell tudni?

- **Destruktív puzzle játék** - rombolj meg mindent!
- **Physics engine** - minden dolog realisztikusan működik
- **Kreatív megoldások** - több módja van megoldani a feladatokat
- **Vizuálisan gyönyörű** - voxel-alapú grafikák
- **Egyedi koncepció** - készülj a lepetésre!

## ⚙️ Telepítés

1. Töltsd le a játékot
2. Csomagold ki a letöltött fájlt
3. Futtasd a `Teardown.exe` (Windows) vagy `Teardown` (Linux) fájlt
4. Kezdd el a rombolást! 💣
