# Among Us

## 📥 Letöltés

Letöltési linkeket a [downloads.txt](downloads.txt) fájlban találod.

## 🖥️ Rendszerre vonatkozó követelmények

### Windows
- **OS:** Windows 7 SP1 vagy újabb
- **Processzor:** Intel Core 2 Duo E8400
- **RAM:** 1 GB
- **Videokártya:** Intel UHD Graphics 630

### Linux
- **OS:** Ubuntu 16.04 LTS vagy újabb
- **Processzor:** Intel Core 2 Duo E8400
- **RAM:** 1 GB
- **Videokártya:** Intel UHD Graphics 630

## 🎮 Mit kell tudni?

- **Casual játék** - könnyen játszható, szórakoztató
- **Multiplayer** - csak több játékossal érdemes játszani
- **Gyors mechaniká** - egyenként 10-20 percig tartanak a körök

## ⚙️ Telepítés

1. Töltsd le a játékot
2. Csomagold ki a letöltött fájlt
3. Futtasd az `AmongUs.exe` (Windows) vagy `AmongUs` (Linux) fájlt
4. Élvezd a játékot! 🎉
