## 🎮 Mit kell tudni?

- **2D sandbox játék** - építs, fedezz fel, harcolj!
- **Kreatív játékmód** - végtelen lehetőségek
- **Multiplayer** - játssz barátaiddal vagy egyedül
- **Bossok** - Megafon és képernyőn kívüli kihívások

## ⚙️ Telepítés

1. Töltsd le a játékot
2. Csomagold ki a letöltött fájlt
3. Futtasd a `Terraria.exe` (Windows) vagy `Terraria` (Linux) fájlt
4. Légy szórakoztató! ⛏️
